# ■ VentureScope
> Can we predict whether a startup will reach IPO, get acquired, or shut down —
> using only its funding history and investor profile?
## Stack
- Python (pandas, numpy) — data cleaning and feature engineering
- Databricks Community Edition — cloud compute and Delta Lake storage
- dbt — SQL transformation models (staging + marts)
- Spark SQL — analytical queries
## Data Source
Crunchbase startup investments dataset via Kaggle.
kaggle.com/datasets/arindam235/startup-investments-crunchbase
